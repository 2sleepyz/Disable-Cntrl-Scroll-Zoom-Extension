# Disable-Cntrl-Scroll-Zoom-Extension
A extension i made to disable chrome's cntrl+scroll feature including the - , + , = , 0 features.
# Instructions
- first enable devloper mode in chrome://extensions
- then press load unpacked and choose the disable-ctrl-scroll-zoom folder
- then a new extension will appear then flick it on
- then pin it in ur extensions dropdown
- then toggle it on and off.
- have fun
